Step 1: Lyα 谱线检测

在光谱蓝端（< 4000 Å）流量较大、且在大 smoothing 尺度（sigma=16）下可见的宽峰中，候选 Lyα 谱线有 2673.759 Å、2836.879 Å 和 3163.121 Å。其中，2673.759 Å 的峰虽然流量（0.750）略低于其他两条，但其在 sigma=16 的全局平滑尺度上被识别，表明其是一个显著的大尺度结构。相比之下，2836.879 Å 和 3163.121 Å 主要在局部平滑尺度上被识别。综合考虑显著性和位置，最可能的 Lyα 候选线是 **2673.759 Å**。

- 观测波长 λ_obs: 2673.759 Å
- 流量 Flux: 0.750
- 谱线宽度: 37.814 Å

该峰作为 Lyα 发射线对应的红移 z 为 **1.199**。

关于 Lyα forest 的检查：在该候选 Lyα 线（2673.759 Å）的蓝端（即波长 < 2673.759 Å），存在多条吸收线，例如在 2468.085 Å、2212.766 Å 和 2007.092 Å 处。这些吸收线相对密集、较窄（宽度大多小于 70 Å），并且集中在 Lyα 峰的短波长一侧，符合高红移 QSO 光谱中由中性氢云产生的 Lyα forest 特征。这进一步支持了 z ≈ 1.2 的红移估计。

Step 4: 补充步骤（假设 Step 1 所选择的谱线并非 Lyα）

- **假设峰值 2673.759 Å 对应 C IV (λ_rest = 1549.48 Å)**
    - 观测波长 λ_obs: 2673.759 Å
    - 流量 Flux: 0.750
    - 谱线宽度: 37.814 Å
    - 初步计算红移 z: 0.726

    在此红移下，其他主要发射线的理论观测位置为：
    - Lyα (1215.67 Å): 2097.742 Å
    - C III] (1908.73 Å): 3293.676 Å
    - Mg II (2798.75 Å): 4829.481 Å

    检查光谱匹配情况：
    - **Lyα**: 理论位置 2097.742 Å 位于光谱蓝端（< 4000 Å），该区域信噪比低。在 2007.092 Å 和 2212.766 Å 处存在吸收线，但未见显著的、与 Lyα 预期强度和宽度匹配的**发射峰**。最近的强发射峰在 2312.057 Å，与理论值偏差较大。
    - **C III]**: 理论位置 3293.676 Å。在 3163.121 Å 和 3326.241 Å 处有发射线，但它们与理论值相差超过 100 Å，且 3326.241 Å 的流量和宽度均较小，不像是主宽线。
    - **Mg II**: 理论位置 4829.481 Å。在 4758.865 Å 处存在一个极强的宽发射线（Flux=5.917, Width=74.147 Å），这是光谱中最显著的特征之一。然而，其位置与 C IV 推断出的 Mg II 位置相差约 70 Å，超出了典型的波长误差范围（±53 Å），匹配不佳。

    结论：若将 2673.759 Å 解释为 C IV，则预期的 Mg II 发射线（应在 ~4829 Å）与实际最强的宽线（在 4759 Å）无法对齐，且 Lyα 和 C III] 也缺乏明确对应。因此，该假设**不合理**。

- **尝试其他假设：考虑最强峰 4758.865 Å 为主发射线**
    由于 4758.865 Å 处的发射线是光谱中流量最高、最宽且在全局平滑尺度上显著的特征，更可能是由 Mg II 或 C III] 产生的。

    - **假设 4758.865 Å 对应 Mg II (λ_rest = 2798.75 Å)**
        - 计算得红移 z = 0.700。
        - 在此红移下，C III] (1908.73 Å) 的预期位置为 3245.727 Å，Lyα (1215.67 Å) 的预期位置为 2066.826 Å。
        - 检查 C III] 区域：在 3163.121 Å 处有一个显著的宽发射线，与预期的 3245.727 Å 相差约 83 Å，偏差较大。
        - 检查 Lyα 区域：同样，在 2067 Å 附近无显著发射峰，最近的强峰在 2312 Å。

    - **假设 4758.865 Å 对应 C III] (λ_rest = 1908.73 Å)**
        - 计算得红移 z = 1.493。
        - 在此红移下，Mg II (2798.75 Å) 的预期位置为 6977.284 Å，Lyα (1215.67 Å) 的预期位置为 3030.665 Å。
        - 检查 Mg II 区域：在 7439.716 Å 和 7709.220 Å 处有发射线，但与 6977 Å 的预期位置偏差过大（>450 Å）。
        - 检查 Lyα 区域：在 3031 Å 附近，有 3163.121 Å 和 2836.879 Å 的发射线，但均不精确匹配。

- **寻找自洽的多线匹配**
    尝试将光谱中几个主要宽发射线直接与静止波长关联，计算各自的红移：
    - Lyα--1215.67--2312.057--0.902
    - C III]--1908.73--3163.121--0.657
    - Mg II--2798.75--4758.865--0.700

    这些独立计算的红移值（0.902, 0.657, 0.700）彼此不一致，没有形成一个统一的红移解。这表明，除了最初假设的 Lyα 在 2673.759 Å（z=1.199）外，很难用其他单一红移来同时解释多个主要宽发射线的位置。考虑到 Lyα forest 的存在以及 2673.759 Å 峰的全局显著性，最初的 Lyα 假设仍然是最自洽的解释。其他假设均因缺乏多条发射线的一致性支持而显得不合理。
