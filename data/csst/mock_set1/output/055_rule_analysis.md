Step 1: Lyα 谱线检测

在蓝端流量较大、大 smoothing 尺度（sigma=16）下可见且具有一定宽度的峰中，候选 Lyα 谱线包括：2617.021 Å, 3205.674 Å, 3418.440 Å, 3914.894 Å, 4099.291 Å, 4787.234 Å。

分析这些候选线：
- 所有候选线均在大尺度平滑（sigma=16）下被识别，符合强而宽的 Lyα 特征。
- 其中，4787.234 Å 的流量（Flux=3.505）和显著度（Prominence=3.891）远高于其他候选线，是光谱中最突出的发射线。其宽度为 88.298 Å。
- 虽然 2617.021 Å 位于更蓝端，但其流量（Flux=0.675）和显著度（Prominence=0.764）明显更低，且该区域（2000-4500 Å）信噪比低、数据不可靠。

因此，最可能的 Lyα 发射线是 **4787.234 Å**。

- 观测波长 λ_obs: 4787.234 Å
- 流量 Flux: 3.505
- 谱线宽度: 88.298 Å

当此峰为 Lyα (λ_rest = 1216 Å) 时，计算得到的红移 z 为 **2.937**。

关于 Lyα forest 的检查：
在推测的 Lyα 线（4787.234 Å）的蓝端（即波长 < 4787 Å），尤其是在 3992.908 Å, 4567.376 Å, 4212.766 Å, 3695.035 Å, 3588.652 Å 等处存在多条吸收线。这些吸收线相对发射线更窄，且在 Lyα 蓝端形成了一定的“密集”分布趋势，符合高红移 QSO 中由中性氢云造成的 Lyα forest 的基本特征。尽管由于整体信噪比较低，forest 结构不够清晰，但其存在性与 z≈2.94 的红移预期相符。

Step 2: 其他显著发射线分析

1. 基于 Step 1 得到的红移 z = 2.937，计算三条主要发射线的理论观测波长：
   - C IV 1549 Å 的理论位置为 6098.413 Å。
   - C III] 1909 Å 的理论位置为 7515.733 Å。
   - Mg II 2799 Å 的理论位置为 11019.663 Å，该位置已远超光谱的红端截止波长（8007.092 Å），因此在本光谱中不可见。

2. 在提供的光谱峰值列表中检查匹配情况：
   - 在 C IV 理论位置 6098.413 Å 附近，存在一个显著的宽发射线，其观测波长为 6113.475 Å。该峰在最大平滑尺度（sigma=16）下被识别，流量和显著度均较高，与 C IV 的预期特征相符。
   - 在 C III] 理论位置 7515.733 Å 附近，存在一个观测波长为 7503.546 Å 的宽发射线。该峰同样在最大平滑尺度下被识别，是光谱红端最突出的发射线之一，与 C III] 的预期相符。
   - Mg II 2799 Å 因红移到 11019.663 Å 而落在光谱范围之外，故无对应峰。

3. 根据匹配的观测峰值，重新计算各发射线对应的红移：
   - C IV--1549--6113.475--2.947
   - C III]--1909--7503.546--2.931

Step 3: 综合判断
C IV 和 C III] 两条主要谱线均已被成功识别，且位置与基于 Lyα 推测的红移基本一致。各谱线计算出的红移分别为：Lyα (z=2.937)，C IV (z=2.947)，C III] (z=2.931)。三者差异在可接受范围内，表明红移解自洽。

根据规则，应采用最低电离态谱线的红移作为最终结果。在 Lyα、C IV、C III] 中，Mg II 是更低电离态的谱线，但其已红移出光谱范围。在可见的匹配谱线中，C III] (1909 Å) 的电离态低于 C IV (1549 Å)，而 Lyα 虽为中性氢，但因其易受外流和阻尼翼影响，置信度较低，故不优先采用。

因此，在高置信度的金属线中，C III] 代表最低电离态，其对应的红移为 **2.931**。

Step 4: 补充步骤（假设 Step 1 所选择的谱线并非 Lyα）

- **假设峰值 4787.234 Å 对应 C IV (λ_rest = 1549 Å)**
    - 观测波长 λ_obs: 4787.234 Å
    - 流量 Flux: 3.505
    - 谱线宽度: 88.298 Å
    - 初步计算红移 z: 2.091

    在此红移下，其他主要发射线的理论观测位置为：
    - Lyα (1216 Å): 3758.656 Å
    - C III] (1909 Å): 5900.719 Å
    - Mg II (2800 Å): 8654.800 Å

    检查光谱匹配情况：
    - **Lyα**: 理论位置 3758.656 Å。在该位置附近（如 3914.894 Å, 4099.291 Å）存在强发射线，但与 3758 Å 偏差较大（>150 Å），且 3758 Å 附近无显著峰。因此，未发现与 C IV 假设下 Lyα 位置匹配的发射线。
    - **C III]**: 理论位置 5900.719 Å。在 6113.475 Å 处存在一条强而宽的发射线（Flux=1.273, Width=209.818 Å），两者相差约 213 Å，超出波长误差范围（±53 Å），不构成良好匹配。
    - **Mg II**: 理论位置 8654.800 Å，已超出光谱红端极限（8007 Å），不可见。

    结论：若将 4787.234 Å 峰视为 C IV，则预期的 C III] 和 Lyα 发射线在光谱中均无可靠对应，该假设不合理。

- **假设峰值 4787.234 Å 对应 C III] (λ_rest = 1909 Å)**
    - 计算红移 z: (4787.234 / 1909) - 1 ≈ 1.508
    - 在此红移下，Lyα 的理论位置为 1216 * (1 + 1.508) ≈ 3050.0 Å。该区域（2000-4500 Å）信噪比极低，数据不可靠。虽然在 3205.674 Å 和 3418.440 Å 有候选峰，但其流量和显著度远低于主峰，无法确认为 Lyα。
    - Mg II 的理论位置为 2800 * (1 + 1.508) ≈ 7022.4 Å。在 6744.681 Å 和 7503.546 Å 有微弱发射线，但与 7022 Å 偏差较大，且强度很弱，匹配不佳。

    此外，检查其他强发射线以寻找自洽的红移：
    - 候选 Lyα: 4099.291 Å → z(Lyα) = 2.371
    - 候选 C III]: 6113.475 Å → z(C III]) = 2.202
    - 若 4099.291 Å 是 C IV，则 z = 1.646；若 6113.475 Å 是 Mg II，则 z = 1.183。这些红移值彼此间差异巨大，无法形成一个统一的、能同时解释多条谱线的红移解。

综合来看，所有将 4787.234 Å 峰解释为非 Lyα 主要发射线（如 C IV 或 C III]）的假设，都无法在光谱中找到其他关键发射线（特别是 Lyα）的可靠对应，导致红移解不自洽。这反过来强有力地支持了 Step 1 的原始结论，即 4787.234 Å 最合理的身份就是 Lyα 发射线，对应的红移 z ≈ 2.937。
