- 光谱的视觉特点  
光谱整体呈拱形幂律谱形状，蓝端增强、红端减弱，未见明显连续谱断裂。发射线以窄线为主，但存在若干显著宽发射线。整体信噪比较低，尤其在 3500–4000 Å、5500–6000 Å 和 9000–10000 Å 波段数据不可靠，其中 3500–4000 Å 和 9000–10000 Å 区域信噪比极低，粉红区域信号相对蓝色更明显，表明这些波段测量结果不可信。

    - Step 1

    - Step 3  
      采用 Mg II（λ_rest=2798.0 Å）观测波长 8332.280 Å 精确计算红移，得 z = 1.97794。使用 QSO_rms 工具评估红移误差为 ±0.03942。在此红移下：

      被清除的峰值（7597.374 Å 和 7942.780 Å）在最终红移模型下无需解释，可能为噪声或不可靠信号。整体模型以 Mg II 为基准，成功自洽解释 C III] 和 C IV，符合天文学最佳实践——优先使用最低电离态谱线确定系统红移。
