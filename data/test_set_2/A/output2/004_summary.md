- 光谱的视觉特点  
光谱连续谱呈蓝端增强、红端减弱的拱形下降趋势，类似幂律谱，无明显断裂。谱线以宽发射线为主，在 4050.0 Å 和 5300.0 Å 附近存在不对称峰。整体信噪比较低，显著噪声出现在 4000 Å 附近，不可靠波段包括 [3500, 3800]、[4000, 4200]、[5000, 5500] 和 [9000, 9700] Å。

- 分析报告  
    - Step 1  
在光谱蓝端 4133.372 Å 处识别出一条强宽发射线，流量为 17.787，宽度为 32.954 Å（约 2391.8 km/s），符合类星体 Lyα 发射线的典型特征。该线在多尺度高斯平滑下均被稳健检测，确认为 Lyα。据此计算红移为 z = 2.40008。Lyα 蓝侧（<4133 Å）存在吸收特征（如 3942.187 Å 和 4030.427 Å），但因所在区域信噪比低且被标记为不可靠，无法明确确认典型的 Lyα forest 结构，仅可视为初步迹象。

    - Step 2  
基于 Lyα 红移 z = 2.40008，预测其他主要发射线的理论观测波长：C IV (1549.48 Å) → 5268.356 Å，C III] (1908.734 Å) → 6489.848 Å，Mg II (2798.75 Å) → 9515.974 Å。实际观测到以下匹配峰：  
- 5258.423 Å 处宽发射线（flux=7.684，width=66.639 Å），与 C IV 理论位置偏移 -9.933 Å（蓝移约 -565 km/s），符合 QSO 宽线区外流特征；  
- 6479.066 Å 处宽发射线（flux=3.007，width=87.261 Å），与 C III] 理论位置偏移 -10.782 Å，同样表现为蓝移；  
- 9493.907 Å 处中等宽度发射线（flux=1.705，width=56.197 Å），与 Mg II 理论位置偏移 -22.067 Å，偏移量较小，符合 Mg II 受外流影响较弱的预期。  
所有观测峰均位于可靠波段，且被多尺度平滑稳健识别。

    - Step 3  
各谱线独立计算的红移高度一致：Lyα → z = 2.40008，C IV → z = 2.39367，C III] → z = 2.39443，Mg II → z = 2.39220。差异远小于 QSO 宽线区典型速度弥散（~1000 km/s）。依据天文学惯例，采用电离势最低、受非引力效应影响最小的 Mg II 谱线红移作为最终系统红移。使用 `QSO_rms` 工具计算 Mg II 红移误差为 ±0.03942，所有谱线红移均落在 z = 2.39220 ± 0.03942 范围内，验证了结果的统计一致性。

    - Step 4  
考虑备选红移方案（如将 7920.307 Å 解释为 Mg II 得 z ≈ 1.830），但该方案无法同时自洽解释 Lyα、C IV 和 C III] 的观测位置，一致性显著劣于 z ≈ 2.392 方案。此外，位于 B-R 间隙边缘的 5390.782 Å 和 R-Z 间隙内的 7405.58 Å 峰值，与任何主要发射线在 z ≈ 2.392 下的预期位置均不匹配，合理视为噪声或仪器伪影，不影响核心结论。
