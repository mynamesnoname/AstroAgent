Step 1: Lyα 谱线检测

在光谱蓝端（短波长区域），流量较高，且经大尺度（sigma=16）平滑后仍可见的宽发射线中，最可能为 Lyα 的是观测波长为 4405.443 Å 的谱线。理由如下：
- 该线是所有候选中最靠蓝端的，符合 Lyα 通常作为最显著且位于紫外蓝端的强发射线特征。
- 其宽度为 55.570 Å（对应约 3784 km/s），被标记为“宽线”，这与 QSO 中 Lyα 发射线通常具有较宽轮廓（受中心黑洞引力及气体运动影响）的特性一致。
- 尽管 6023 Å 处的线最强，但其红移高达 z≈3.95，会将许多其他重要发射线（如 C IV 1549, C III] 1909）推至远红端甚至超出本光谱范围（9817 Å），而 z≈2.62 更能合理覆盖常见的 QSO 发射线系。

因此，选择 λ_obs = 4405.443 Å 作为 Lyα 候选。

- **观测波长 λ_obs**: 4405.443 Å
- **流量 Flux**: 0.751
- **谱线宽度**: 55.570 Å (3784.156 km/s)
- **对应红移 z**: 2.623

关于 Lyα forest 的检查：
在所选 Lyα 线（4405.443 Å）的蓝端（即波长 < 4405 Å），光谱数据从 3604 Å 开始。在此区域内，我们观察到多条窄的吸收特征，例如在 4111.313 Å、3898.068 Å 和 3670.116 Å 处的吸收线。这些吸收线相对密集、较窄，且集中在 Lyα 发射线的短波长一侧，符合高红移 QSO 光谱中由中性氢云产生的 Lyα forest 的典型特征。尽管信噪比较低且 3500-4000 Å 区域被标记为不可靠，但这些吸收结构的存在为 z ≈ 2.62 的红移解提供了额外的支持。

2. 将理论位置与观测到的发射线进行比对：
   - **C IV 1549**: 理论位置 5612.027 Å。在该位置附近，光谱中存在一条显著的宽发射线，其观测波长为 5214.303 Å（宽线，流量 0.454），但与理论值偏差较大（~400 Å）。更值得注意的是，在 6023.163 Å 处存在一条非常强的宽发射线（流量 0.933），这可能是被红移得更高的 C IV 线。
   - **C III] 1909**: 理论位置 6916.307 Å。在该位置附近，光谱中存在一条窄发射线，观测波长为 7155.567 Å（流量 0.472）。两者相差约 239 Å，在波长误差（±55 Å）和 QSO 谱线内在速度偏移（通常可达数百 km/s）的合理范围内，可以认为是匹配的。
   - **Mg II 2799**: 理论位置 10140.777 Å，已超出本光谱的红端极限（9817.451 Å），因此无法在数据中观测到。

Step 4: 补充步骤（假设 Step 1 所选择的谱线并非 Lyα）

- **假设该峰值（λ_obs = 4405.443 Å）为 C IV 1549 Å 发射线**：
    - **观测波长 λ_obs**: 4405.443 Å
    - **流量 Flux**: 0.751
    - **谱线宽度**: 55.570 Å (3784.156 km/s)
    - **对应红移 z**: 1.844

    在此红移下，其他主要发射线的理论观测位置为：
    - Lyα (1215.67 Å): 3457.365 Å
    - C III] (1909.0 Å): 5429.196 Å
    - Mg II (2798.0 Å): 7957.512 Å

    检查光谱匹配情况：
    - **Lyα**: 理论位置 3457.365 Å 位于光谱起始波长（3603.937 Å）之前，因此不在观测范围内，无法检验。
    - **C III]**: 理论位置 5429.196 Å 附近存在一条显著的发射线（λ_obs = 5471.668 Å, Flux = 0.390），其轮廓为中等宽度。两者波长差约为 42.5 Å，在给定的波长误差（~±55 Å）范围内，可以认为是合理的匹配。
    - **Mg II**: 理论位置 7957.512 Å 附近存在一条强窄发射线（λ_obs = 8008.546 Å, Flux = 1.491）。两者波长差约为 51.0 Å，同样在误差允许范围内，构成良好匹配。

    基于这些匹配，计算各线对应的红移：
    - C III]--1909.000--5471.668--1.866
    - Mg II--2798.000--8008.546--1.862

    这两个独立计算的红移值（1.866 和 1.862）与基于 C IV 假设的红移（1.844）非常接近，差异在 0.02 以内，考虑到谱线宽度和测量误差，这是一致的。因此，将 4405.443 Å 的谱线解释为 C IV 是一个自洽且合理的假设。

- **假设该峰值为 C III] 1909 Å 发射线**：
    - 对应红移 z = 1.308。
    - 在此红移下，Lyα 应出现在 2805.766 Å（超出光谱蓝端），Mg II 应出现在 6457.784 Å。
    - 在 6457.784 Å 附近，光谱中确实存在一条窄发射线（λ_obs = 6677.604 Å），但两者相差约 220 Å，远超波长误差范围，匹配不佳。
    - 此外，若红移为 1.308，则 C IV 应出现在约 3573 Å，该区域数据不可靠且无显著宽发射线特征。
    - 因此，将 4405.443 Å 解释为 C III] 的假设与光谱中其他特征不吻合，不合理。

综上所述，虽然 Step 1 将 4405.443 Å 解释为 Lyα 并得出 z≈2.62 是一个可行方案，但将其解释为 C IV 并得出 z≈1.86 同样是一个高度自洽的解，并且能得到 C III] 和 Mg II 发射线的良好支持。两种红移解均需考虑，但 z≈1.86 的解因有多条强发射线相互印证而显得尤为稳健。
